## Prerequisites

**Backend**

- Java 17+
- Maven/Gradle
- MySQL

**Frontend**

- Angular 20.3.6
- Tailwind CSS 4

**Infrastructure**

- Docker & Docker Compose
- MinIO (S3-compatible storage)
